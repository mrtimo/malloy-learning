A Basic Checklist for Beginners

The following steps will help you get you up and running with your data and Malloy in VSCode

  1. Download your .csv file
  2. Rename it to be something short and simple (no spaces) like ev.csv or cars.csv or aud.csv
  3. Put it in a folder on your computer (not one drive)
  4. Open the folder in VSCode (File -> Open Folder).
  5. Click on the folder you want to open, then press ‚ÄúOpen‚Äù
  6. You should have already installed the Malloy Extension and Rainbow CSV extensions - but if not, click the lego block icon
  7. Click the file icon (at the top on the left)
  8. Create a new (.malloynb) file - name is something simple like cars.malloynb
  9. If you have the extension installed your file will have a malloy icon next to it like this:
  10. Click into your file and press the button that says +Code
  11. This is where you can write your source: statement like you have seen before.
  12. Make sure you can press schema or preview on your source before going forward.
    1. Make sure you have typed the name of the .csv file correctly.
  13. If you are having issues with your source statement here are some common problems:
    1. Your data is on one drive, and not on your main computer drive.¬†
      1. To fix this, move the folder to your Documents or Downloads or Desktop folder
    2. The CSV file is not UTF-8 encoded. (Google UTF-8 incoding for more information)
      1. UTF-8 encoding is is most popular and widely used format for character encoding.¬†
      2. The best way to encode the file properly is to open in Excel, press Save As, and in the save as select the CSV option with UTF-8 encoding.
